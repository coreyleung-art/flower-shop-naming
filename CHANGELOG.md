## [2.19.0] - 2026-04-19

### 修复
- **百合品种提取顺序**：修复具体百合品种（如西伯利亚、卡萨布兰卡）被大类名称（东方百合）先匹配的问题

### 使用方式
```bash
# 从 Notion 加载数据
python3 item_naming_cli.py "卡罗拉玫瑰" --no-cache

# 强制刷新缓存
python3 item_naming_cli.py "卡罗拉玫瑰" --force-refresh

# 默认使用缓存
python3 item_naming_cli.py "卡罗拉玫瑰"
```
